Here's a complete step-by-step guide to create and set up a GitHub App:

### Create GitHub App:
   - Go to your organization settings
   - Click on "Developer settings" (bottom of left sidebar)
   - Click "GitHub Apps"
   - Click "New GitHub App"
   - Fill in these details:
     ```
     Name: Fumodocs Feedback Bot
     Homepage URL: [Your website URL]
     Webhook: Disable
     ```

### Set App Permissions:
   - Under "Repository permissions":
     - Discussions: Read & Write
     - Contents: Read & Write
   - Under "Organization permissions":
     - Members: Read-only
   - Under "User permissions":
     - None required

### Save and Get Credentials
   - Click "Create GitHub App"
   - You'll get:
     - App ID (save this)
     - Generate a private key (download it)
     - Keep both secure

### Install the App:
   - Go to your GitHub App settings
   - Click "Install App"
   - Select your organization
   - Choose repositories to install on
   - Click "Install"

### Set Environment Variables:
   - In your `.env.local`:
     ```
     GITHUB_APP_ID=your_app_id
     GITHUB_APP_PRIVATE_KEY="-----BEGIN RSA PRIVATE KEY-----\nYour private key here\n-----END RSA PRIVATE KEY-----"
     ```
